# Shopping Cart

點飲料的購物車系統

## Features

- 飲料選擇：展示各種飲料選項，讓使用者可以從列表中選擇飲料。
- 客製化需求：提供使用者選擇不同的甜度、冰塊量、加料等參數。
- 即時總計：在確認訂單前即時顯示總計金額。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
